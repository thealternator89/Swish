⚠️ Warning

When used safely, Node Modules can be incredibly useful - doing a lot of work for your plugin, but if you install the wrong one they might cause problems.

Take care when installing modules, and consider the following before installing:

  • Are many other people using it?
  • Has it been around for a while, or is it new?
  • Is it from a trusted source?
  • Do other popular modules use it?
