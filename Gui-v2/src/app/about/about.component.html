This is a prerelease version of Swish v2.
Swish is an awesome data manipulation tool which runs on your desktop, terminal, and in the browser.
Swish runs on JavaScript. This version is built with Electron and Angular. Plugin icons are from Material Design.
This is just one of several apps built on the Swish platform, including:
  • Swish Core: The raw experience of Swish. Runs in the terminal and is ready for anything.
  • Swish Clip: Runs in the background, always ready to manipulate your clipboard.
  • Swish GUI v1: The original.
  • Swish GUI v2: The next generation of Swish. ⇐ This is what you're currently using.
  • Swish Web: Hosted Swish experience. Host it somewhere you trust and Swish without even installing it.
Please Note: Due to the changes made in Swish GUI v2, Swish Clip and Swish GUI v1 are not guaranteed to continue working with future plugins once Swish v2 is released.
Swish Core and Web will be updated as part of the Swish v2 release. Swish Clip and original GUI will continue to work with plugins built for Swish v1.
